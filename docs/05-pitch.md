## Sobre o Pitch

Apresentação de 3 minutos cobrindo:

- **O problema:** Estudantes de tecnologia sem orientação personalizada para tomar decisões de carreira
- **A solução:** MentorIA, um agente inteligente que conhece o perfil do usuário e recomenda trilhas, certificações e planos de estudo com base em dados reais
- **A demonstração:** Interação ao vivo com o chatbot via Streamlit, mostrando personalização e anti-alucinação na prática
- **O impacto:** Democratizar o acesso à mentoria de carreira para quem está começando na área de tecnologia

---

## Tecnologias apresentadas

- **LLM:** Llama 3.3 70b via Groq API
- **Interface:** Streamlit
- **Base de conhecimento:** JSON + CSV com perfil, histórico de estudos e trilhas disponíveis
